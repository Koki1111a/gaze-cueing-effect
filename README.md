# Gaze Cueing Effect (視線手がかり効果)

## フォルダ構成

### assets
実験用の刺激画像が入ったフォルダ

### data
実験結果のデータ

### mypackage
直接実行しないプログラム

## 資料

### 実験内容.py
実験の背景や内容について記載

## 実行プログラム

**(ダブルクリックでスタート)**  
※ 「Ctrl+C」を押してから，「L」を押すなどして次の画面に進めるとプログラムが強制的に終了します。

- **experiment_A.py**  
  実験A

- **experiment_B.py**  
  実験B

- **test.py**  
  デバッグ用

- **trail.py**  
  被験者の練習用
