# Gaze Cueing Effect (視線手がかり効果)

## フォルダ構成

### assets/
実験用の刺激画像が入ったフォルダ

### data/
実験結果のデータ

### mypackage/
直接実行しないプログラム

## 資料

### 実験内容.py
実験の背景や内容について記載

## 実行プログラム

※ 「Ctrl+C」を押してから，「L」を押すなどして次の画面に進めるとプログラムが強制的に終了します。

- 実験A
```bash
python experiment_A.py
```

- 実験B
```bash
python experiment_B.py
```

- デバッグ用
```bash
python test.py
```

- 被験者の練習用
```bash
python trail.py
```
